## 🌟 Key Features & Focus
### 1. Responsive Design & UX
- **Fluid UI:** `clamp` 함수와 Tailwind의 반응형 접두사를 활용하여 모바일(320px)부터 데스크탑(1920px)까지 끊김 없는 사용자 경험을 제공합니다.
- **Micro-Interactions:** Lottie와 CSS 애니메이션을 결합하여 자칫 지루할 수 있는 포트폴리오에 생동감을 더했습니다.

### 2. Scalable Component Structure
- **Data-Driven UI:** 프로젝트와 기술 스택 데이터를 별도의 파일로 분리하여, 새로운 내용 추가 시 코드 수정 없이 데이터 확장만으로 반영 가능한 구조를 설계했습니다.
- **Reusable Cards:** `Props`를 활용한 컴포넌트화를 통해 기술 스택과 프로젝트 카드를 공통 모듈로 관리합니다.

### 3. Dynamic Routing
- **Project Detail Page:** `projectId`를 활용한 동적 라우팅을 설계하여, 각 프로젝트의 상세 트러블 슈팅과 핵심 성과를 독립된 페이지에서 효과적으로 전달합니다.
